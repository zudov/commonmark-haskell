isLineEnding :: Char -> Bool Source #

Predicate for line ending character (newline or carriage return). NB: something like `satisfy isLineEnding` won't properly parse a line ending, when given '\r\n' as input it would just consume '\r' leaving '\n' unconsumed. In such cases one should use pLineEnding instead
